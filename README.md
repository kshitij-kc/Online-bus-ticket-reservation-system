# Online-bus-ticket-reservation-system
The Online Bus Ticket Reservation System is a web application that simplifies bus ticket booking. Utilizing HTML, CSS, and JavaScript for the front-end, it provides an intuitive interface for users to search buses, view schedules, and make reservations. The back-end relies on MySQL for efficient database management of routes and user data. The system features content-based filtering for personalized recommendations and implements dynamic pricing to adjust ticket costs based on demand and competition, enhancing user experience and operational efficiency.
